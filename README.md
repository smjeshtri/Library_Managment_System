Project: Library Managment System

This is a university project developed to facilitate and enhance access to library resources within an academic environment. An efficient solution is to digitalize the Library Management System, bringing it closer to all university members while providing better management tools for staff and administrators. This web application allows university members to search for books, borrow and return them, and pay overdue fines, while staff roles handle the daily operations, finances, inventory, and facility maintenance. Members can also leave reviews and feedback on books they have read.
All users access the system through role-based permissions, ensuring that each party only interacts with the features relevant to their responsibilities. Our work consists of taking all the necessary steps that will result in the creation and maintenance of this fully functional  university library web application.
